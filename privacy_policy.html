오늘도 신선 개인정보 처리방침

최종 업데이트: 2025년 1월 31일

1. 개인정보의 수집 및 이용

오늘도 신선(이하 "본 앱")은 사용자의 개인정보를 수집하거나 외부 서버로 전송하지 않습니다. 모든 데이터는 사용자의 기기 내부에만 저장됩니다.

2. 권한 사용 목적

본 앱은 다음과 같은 권한을 사용합니다:

  • 카메라 권한: 식품 유통기한을 촬영하여 자동으로 인식하는 OCR 기능을 제공하기 위해 사용됩니다. 촬영된 이미지는 기기 내에서만 처리되며 외부로 전송되지 않습니다.
  • 저장소 권한: 백업 파일을 저장하거나 불러오기 위해 사용됩니다. 모든 백업 파일은 사용자가 직접 관리합니다.
  • 알림 권한 (Android 13 이상): 유통기한이 임박하거나 만료된 식품에 대한 알림을 표시하기 위해 사용됩니다.

위 권한은 각 기능 사용 시 사용자의 동의를 받으며, 거부하더라도 해당 기능을 제외한 앱의 다른 기능은 정상적으로 사용할 수 있습니다.

3. 데이터 저장 및 보안

본 앱에서 생성되는 모든 데이터는 다음과 같이 관리됩니다:

  • 식품 정보, 유통기한, 메모 등은 기기 내부의 로컬 데이터베이스에만 저장됩니다.
  • 어떠한 데이터도 외부 서버로 전송되지 않으며, 인터넷 연결 없이도 앱을 완전히 사용할 수 있습니다.
  • 백업 기능을 사용할 경우, 사용자가 직접 선택한 위치에 JSON 파일 형태로 저장됩니다.
  • 앱을 삭제하면 저장된 모든 데이터도 함께 삭제됩니다.

4. 제3자 제공

본 앱은 사용자의 개인정보를 제3자에게 제공하지 않습니다.

5. 광고 및 분석 도구

본 앱은 광고를 표시하지 않으며, Google Analytics 등의 분석 도구를 사용하지 않습니다.

6. ML Kit 사용

본 앱은 Google ML Kit의 텍스트 인식 기능을 사용하여 유통기한을 인식합니다. ML Kit는 기기 내에서 동작하며 (On-device processing), 이미지 데이터는 외부로 전송되지 않습니다.

7. 아동의 개인정보 보호

본 앱은 만 13세 미만 아동을 대상으로 하지 않으며, 고의로 아동의 개인정보를 수집하지 않습니다.

8. 개인정보 처리방침 변경

본 개인정보 처리방침은 법령, 정책 또는 보안 기술의 변경에 따라 내용이 추가, 삭제 및 수정될 수 있습니다. 변경 시에는 앱 업데이트를 통해 고지할 예정입니다.

9. 사용자의 권리

사용자는 언제든지 다음과 같은 권리를 행사할 수 있습니다:

  • 앱 내에서 직접 데이터를 조회, 수정, 삭제할 수 있습니다.
  • 앱 설정에서 권한을 변경하거나 철회할 수 있습니다.
  • 백업 기능을 통해 데이터를 내보내거나 가져올 수 있습니다.
